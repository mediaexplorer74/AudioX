# AudioX A free and simple UWP based browser for Mobile (and Desktop) with a range of feature to try to improve browsing on Windows MobileA free and simple cross-platform "audio-subsystem" for Xamarin.Forms 2.5 and Windows 10 Mobile. Based on [https://github.com/adrianstevens/Xamarin-Plugins](https://github.com/adrianstevens/Xamarin-Plugins) solution.## Screenshots![](Images/shot1.png)![](Images/shot2.png)## Recent changes```- .
